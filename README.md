# CodeChef-Task2-NLP

Simple NLP text classification model built using scikit-learn.

- Preprocesses raw text (tokenization, stopword removal, basic cleaning).
- Converts text to numerical features using TF-IDF / Bag-of-Words.
- Trains and evaluates a classifier on the given CodeChef task dataset.
- Implemented and run in a Jupyter/Colab notebook.
